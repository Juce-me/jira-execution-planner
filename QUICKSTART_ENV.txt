QUICK START - New .env Based Setup
===================================

1. INSTALL DEPENDENCIES:
   python3 -m pip install --user python-dotenv

2. CREATE .env FILE:
   cp .env.example .env

3. OPEN .env AND FILL IN:
   nano .env

   Change:
   - JIRA_EMAIL to your email
   - JIRA_TOKEN to your API token
   - JQL_QUERY if needed

4. START SERVER:
   python3 jira_server.py

5. OPEN DASHBOARD:
   Open jira-dashboard.html in browser
